内部类笔记

在类内部定义的类,并且在类中声明成一个成员,那么这个类就是内部类.

没有用static修饰的成员内部类被称之为实例内部类,static修饰的称之为静态内部类.

成员内部类|实例内部类

源码：

OuterClass.java

package InnerClassPratice.inner;

public class OuterClass {
    public class InnerClass{
        public void showInner(){
            System.out.println("执行了OuterClass中的InnerClass内部类的方法");
        }
    }
}

MainDemo.java

package InnerClassPratice.inner;

public class MainDemo {
    public static void main(String[] args) {
        new OuterClass().new InnerClass().showInner();
    }
}

定义方法

参考Outer.java可知,内部类就是一个定义在类内部的类,可以在类内方法中直接创建局部对象.

比如:

    public class Outer1{
        public class Inner {
            public class InnerClass(){
            void ShowMethod(){
                System.out.println("执行了内部类的方法")
            }
            }
        }

        void innerTest(){
        new InnerClass().ShowMethod()
        }
    }

也可以定义一个成员内部类后,建立一个成员对象.

比如

    public class Outer1{
        Inner obj= new Inner();


        public class Inner {
                public class InnerClass(){
                    void ShowMethod(){
                        System.out.println("执行了内部类的方法")
                    }
                }
            }
}

如果过是想在外部类外部创建对象,则需要使用外部类创建一个对象,通过这个对象来创建这个类. 方式如下:

1先创建一个外部类对象

Outer1 obj=new Outer1()

2通过这个对象创建成员内部类

obj.InnerClass innerObj=obj.new InnerClass()

3调用这个类的方法

innerObj.ShowMethod()

连续简洁地一次表达方式如下:

Outer1.InnerClas innerObj= new Outer1().new InnerClas()

成员内部类|静态内部类

源码

OuterClass.java

package InnerClassPratice.inner;

public class OuterClass {
    InnerClass InnerObj=new InnerClass(){
        @Override
        public void showInner(){
            System.out.println("执行了Outerclass中的成员对象的方法");}

    };
    public static class StaticInnerClass{
        static public void showStaticInner(){
            System.out.println("执行静态成员内部类静态方法");
        }
        public void showStaticInnerNotstatic(){
            System.out.println("执行静态成员内部类的非静态方法");
        }
    }
    public class InnerClass{
        public void showInner(){
            System.out.println("执行了OuterClass中的InnerClass内部类的方法");
        }
    }
}

MainDemo.java

package InnerClassPratice.inner;

public class MainDemo {
    public static void main(String[] args) {

        OuterClass.StaticInnerClass.showStaticInner();
        new OuterClass.StaticInnerClass().showStaticInnerNotstatic();
        new OuterClass().new InnerClass().showInner();
        new OuterClass().InnerObj.showInner();

    }
}

从上述代码可见,静态类可以直接通过类名.成员内部类名进行引用

比如:
new OuterClass.StaticInnerClass()
//直接通过类名.内部类名()创建一个新的对象

如果静态内部类成员方法也是静态的,则可以通过类名.成员内部类名.方法名()调用方法

比如:
OuterClass.StaticInnerClass.showStaticInner();

局部内部类

参考局部变量,局部类与局部变量类似,都是在方法体内定义的.

局部内部类就是在类内成员方法中定义的类,其作用域仅在方法体内.

这种内部类的特点:

  1. 生命周期仅仅在其所在方法执行期间
  2. 不能够被权限修饰词和static修饰,这是因为他只能在方法内被访问.

源码:

OuterClass.java

package InnerClassPratice.innerpart;

public class OuterClass {
    void showInnerMethod(){
        class InnerClassPart{
            void show(){
                System.out.println("执行了局部内部类的方法");
            }
        }
        InnerClassPart A=new InnerClassPart();
        A.show();

    }
}

根据上述代码,InnerClassPart类仅仅在成员方法showInnerMethod()中可访问,且执行完毕该方法后,自动消失.

MainDemo.java

package InnerClassPratice.innerpart;

public class MainDemo {
    public static void main(String[] args) {
        OuterClass obj=new OuterClass();
        obj.showInnerMethod();
    }
}

匿名内部类笔记

匿名内部类是定义在方法体或代码块之中的,匿名代表着它并没有一个名称.

这种类一般使用在一次性场合,用完就舍弃,有一种临时性.也就是现定义现创建对象现用的类.

实现思路是将"定义","创建","使用"这几条代码压缩成一条代码.

代码实现:

new 父类名/接口名 (){
重写方法()  //类的定义
}

以上语句实现了一条代码定义了一个类并创建了一个对象 类和对象都没有名字.

匿名内部类是通过扩展某个类或实现某个接口实现其定义的,代码实现方法就是对其方法重写.

new InterfaceClass(){
@Override
    void Method(){
    System.out.println("重写接口方法")
    }
}.Method()

源码：

TestInterface.java 接口

package Extend_Pratice.A3;

public interface TestInterface {
void testMethod();
}

MethodSpace.java 方法存放区

package Extend_Pratice.A3;
public class MethodSpace {

/*main() 方法是static方法 只能访问静态方法和静态变量*/

public static void goSwimming(TestInterface testInterface){
testInterface.testMethod();
}
}

MainDemo.java 主方法实验

package Extend_Pratice.A3;

public class MainDemo {
    public static void main(String[] args) {
        TestInterface swim = new TestInterface() {
            @Override
            public void testMethod() {
            System.out.println("有名字的内部类的方法重写");
        }

        };
        MethodSpace.goSwimming(swim);
        MethodSpace.goSwimming(new TestInterface() {
            @Override
            public void testMethod() {
                System.out.println("匿名内部类的方法重写");
             }
        });
}    
}
